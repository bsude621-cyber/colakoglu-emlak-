# Çolakoğlu Emlak — IG İlk Post (Manifesto Carousel)

8 slaytlık açılış postu. Marka tonunu kuran, pin'lenecek "biz buyuz" duyurusu.

## Üretim

```bash
node export.mjs
```

→ `out/01.png ... 08.png` (1080×1350, IG portrait optimal)

## Önizleme

`slides.html` dosyasını tarayıcıda aç — 8 slaytı dikey scroll ile gör.
Tek slaytı izole görmek için `slides.html?slide=3` gibi parametre ekle.
